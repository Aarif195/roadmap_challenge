# TMDB CLI Tool

A simple Node.js CLI application that fetches movie data from the TMDB API.

## How to Run

Run the command below followed by a movie category:

node index.js <command>

## Available Commands

popular  
now_playing  
top_rated  
upcoming  

## Examples

node index.js popular  
node index.js top_rated  
node index.js now_playing  
node index.js upcoming  

The command will return a list of movie titles from TMDB.
