Unit Converter

A simple Node.js and Vanilla JavaScript application for converting Length, Weight, and Temperature units.
It features a clean UI and handles real-time conversion through a local Node server.

Features

Convert between Length, Weight, and Temperature units

Dynamic dropdown updates based on selected type

Real-time conversion using server-side logic

Responsive, user-friendly interface

Tech Stack

Node.js (HTTP Server)

HTML, CSS, JavaScript (Frontend)
